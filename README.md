# SGLauncher
Simple GTK Launcher

## Features: 

- [x] Show system menu apps
- [x] Search On Web button
- [x] Run In Terminal button 
- [x] Show calc answers

- [ ] have quick access buttons
- [ ] have a simple configuration tool
- [ ] Show Actions


## Build 
Main Program:

```
make
```

standalone Simple GTK Launcher with some features
